## Web Audio Player Web Components

This folder contains a minimal demo of custom elements built around the Web Audio API:

- **`my-audio-player`**: main HTML5 audio player using Web Audio (gain, stereo panner, EQ, analyser).
- **`audio-balance`**, **`audio-equalizer`**, **audio-visualizer`**: child components wired to the player.
- **`my-playlist`**: simple playlist component that controls the player (next/prev, optional shuffle/loop).

To try it, serve the folder with a local web server and open `index.html` in your browser or open it from this link **`web-components-miage-o3c3x0hru-3paccs-projects.vercel.app`** .
